# Developer-Profile-Page
# 🌐 Personal Portfolio Webpage

A modern and responsive **Personal Portfolio Webpage** built using **HTML and CSS**.
This project showcases a personal profile with photo, bio, contact details, and social links in a clean and professional design.

---

## 🚀 Features

* 📱 Fully Responsive Design
* 🖼️ Profile Image Display
* 🧾 Professional Bio Section
* 📞 Contact Information
* 🔗 GitHub & LinkedIn Links
* 🎨 Modern UI with Colors & Fonts

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts

---

## ▶️ How to Run

1. Download or clone the repository:

```bash
git clone https://github.com/your-username/personal-portfolio-webpage.git
```

2. Open the folder

3. Double click **index.html**
   OR open using Live Server

---

## 🌍 Live Demo (GitHub Pages)

To host your project:

1. Go to **Repository → Settings**
2. Click **Pages**
3. Select branch: `main`
4. Click **Save**

Your site will be available at:

```bash
https://your-username.github.io/personal-portfolio-webpage/
```

## 🎨 Customization

* Replace profile image:

```html
<img src="profile.png">
```

* Update your details:

```html
<div class="name">Your Name</div>
<div class="bio">Your Bio</div>
```
